Changelog - LENS additions¶

This file lists the various changes made by the LENS fork to the original repository. In time, some of these might be upstreamed to the original repository. We try to rebase our fork on the original from time to time.

You may be interested in the original Ulm-IQO changelog that tracks the changes made to the parent repository.

New interfaces¶

  • A new interface for scanning excitation spectroscopy (commit)

New hardwares¶

  • Thorlabs flip mounts (commit)

  • Integration of Sirah Matisse laser (commit)

  • Integration of MOGLabs’ cateye laser (commit)

  • Scanning excitation spectroscopy interfuse to use NI cards (commit)

  • Dummy excitation scanner (commit)

New logics¶

  • Power sweeping (or really anything that’s logged through a time series) (commit)

  • Scanning excitation logic to work with the scanning excitation interface hardwares (commit)

New GUIs¶

  • Scanning excitation spectroscopy gui, derived from the spectrometer gui (commit).

Modifications to pre-existing elements¶

  • Modified the scanning probe logic to save in the global metadata the current position, so we have it in most saved files (commit)

  • Some fixes when saving with qdplot_logic (commit)

  • Modification of the time series GUI to allow prevent the autoscaling of the y axis (commit)
