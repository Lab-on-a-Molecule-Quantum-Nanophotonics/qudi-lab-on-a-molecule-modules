Scanning excitation spectroscopy¶

class ScanningExcitationLogic(*args, **kwargs)[source]¶

Bases: qudi.core.module.LogicBase

A logic to pilot an ExcitationScannerInterface.

Copy and paste configuration example:

excitation_scanner_logic:
    module.Class: 'scanning_excitation_spectroscopy.ScanningExcitationLogic'
    options:
        beep: True # default
        watchdog_repeat_time_ms: 500 # default
        save_idle_to_global_metadata: True # save the current idle value to global metadata
    connect:
        scanner: excitation_scanner_hardware

The beep option makes the computer beep when a scan finishes to make sure any low attention-span postdoc notices the end of the acquisition. The watchdog_repeat_time_ms COntrols how often the logic checks on the scanning hardware. The save_idle_to_global_metadata option makes the idle frequency being saved in the global metadata of Qudi.

Initialization

Initialise Base instance. Set up its state machine and initialize ConfigOption meta attributes from given config.
