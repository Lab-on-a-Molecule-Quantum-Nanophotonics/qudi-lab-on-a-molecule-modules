Dummy excitation scanner¶

A simple, stupid excitation scanner for testing purposes. It uses a state machine plotted below.

class ExcitationScannerDummy(*args, **kwargs)[source]¶

Bases: qudi.interface.excitation_scanner_interface.ExcitationScannerInterface, qudi.interface.sampled_finite_state_interface.SampledFiniteStateInterface

Copy and paste configuration example:

  excitation_scanner_hardware:
    module.Class: 'dummy.excitation_scanner_dummy.ExcitationScannerDummy'
    options:
      watchdog_delay: 0.2 # default

Initialization

Initialise Base instance. Set up its state machine and initialize ConfigOption meta attributes from given config.

self : object The object being initialized. name : str Unique name for this module instance. configuration : dict Parameters from the configuration file. callbacks : dict Dictionary specifying functions to be run on state machine transitions.
